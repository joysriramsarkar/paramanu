# **App Name**: পরমাণু জগৎ (Poramanu Jogot)

## Core Features:

- উপাদান নির্বাচন (Upadan Nirbachon): একটি ড্রপডাউন মেনু থেকে উপাদান নির্বাচন করার সুবিধা।
- তথ্যাদি প্রদর্শন (Tathadi Prodorshon): নির্বাচিত উপাদানের নাম, প্রতীক, পারমাণবিক সংখ্যা, ভর সংখ্যা এবং ইলেকট্রন কনফিগারেশন প্রদর্শন করার জন্য তথ্য প্যানেল।
- নিউক্লিয়াস ভিজ্যুয়ালাইজেশন (Nucleus Visualization): প্রোটন এবং নিউট্রন সহ নিউক্লিয়াসের ইন্টারেক্টিভ ২ডি ভিজ্যুয়ালাইজেশন।
- কক্ষপথে ইলেক্ট্রন (Kokkothe Electron): বিভিন্ন কক্ষে ঘূর্ণায়মান ইলেক্ট্রনের এনিমেশন।
- ভাষা পরিবর্তন (Bhasa Poriborton): ব্যবহারকারী নিজের ইচ্ছে মতো ভাষায় সবকিছু দেখতে পারবে।

## Style Guidelines:

- Primary color: গাঢ় নীল (#1A237E) যা গভীরতা এবং বিশ্বাসযোগ্যতা প্রকাশ করে।
- Background color: প্রায় কালো (#121212) একটি আধুনিক, নিমজ্জিত অভিজ্ঞতার জন্য।
- Accent color: হালকা সবুজ (#64FFDA) যা তথ্য এবং ভিজ্যুয়ালাইজেশনের জন্য অ্যাকসেন্ট হিসাবে ব্যবহৃত হবে।
- Body and headline font: 'Inter' sans-serif ফন্ট ব্যবহার করা হবে, যা আধুনিক এবং পরিষ্কার।
- সরল এবং পরিষ্কার আইকন ব্যবহার করা হবে যা সহজেই বোঝা যায়।
- বিষয়বস্তু প্রদর্শনের জন্য একটি পরিষ্কার এবং সুসংগত লেআউট ব্যবহার করা হবে, যা ব্যবহারকারীকে সহজে তথ্য খুঁজে পেতে সাহায্য করবে।
- ইলেক্ট্রনের কক্ষপথে ঘূর্ণনের জন্য সূক্ষ্ম এবং অবিচ্ছিন্ন অ্যানিমেশন ব্যবহার করা হবে, যা দৃশ্যটিকে আকর্ষণীয় করে তুলবে।
